# File Naming Convention — Quick Reference

Print this sheet and keep it at your desk.

---

## The Formula

```
YYYY-MM-DD_Title-Words-Here
```

| Part | Rule | Example |
|---|---|---|
| Year | 4 digits | `2026` |
| Month | 2 digits, zero-padded | `04` (not `4`) |
| Day | 2 digits, zero-padded | `07` (not `7`) |
| Separator | Underscore | `_` |
| Title words | Separated by hyphens | `Board-Meeting-Minutes` |
| Literal hyphen | Double hyphen | `8am--2pm` → "8am-2pm" |

---

## By Content Type

### Announcements
```
YYYY-MM-DD_Short-Description
```
```
2026-04-25_Spring-Clean-Up-Day
2026-01-05_Office-Closed--New-Years-Day
2026-07-01_Road-Closure--Route-15-Detour
```

### Monthly Meeting Agendas
```
YYYY-MM-DD_Month-Board-Meeting-Agenda
```
```
2026-05-04_May-Board-Meeting-Agenda
2026-06-01_June-Board-Meeting-Agenda
```

### Monthly Meeting Minutes
```
YYYY-MM-DD_Month-Board-Meeting-Minutes
```
```
2026-04-07_April-Board-Meeting-Minutes
2026-03-02_March-Board-Meeting-Minutes
```

### Special / Additional Meeting Agendas & Minutes
```
YYYY-MM-DD_Description-Of-Meeting-Agenda
YYYY-MM-DD_Description-Of-Meeting-Minutes
```
```
2026-02-24_Planning-Commission-Meeting-Agenda
2026-03-17_Additional-Meeting--Budget-Review-Minutes
```

### Ordinances
No date needed — just a clear descriptive name, sorted alphabetically on the website.
```
Zoning-Ordinance
Subdivision-and-Land-Development-Ordinance
Floodplain-Ordinance
Stormwater-Management-Ordinance
```

## Zero-Padding Reference

| Month / Day | Correct | Wrong |
|---|---|---|
| January | `01` | `1` |
| February | `02` | `2` |
| March | `03` | `3` |
| April–September | `04`–`09` | `4`–`9` |
| 1st–9th of month | `01`–`09` | `1`–`9` |
| October–December | `10`–`12` | ✓ already 2 digits |
| 10th–31st | `10`–`31` | ✓ already 2 digits |

## Quick Test

Before uploading, ask yourself:

- [ ] Does the date come first?
- [ ] Is the month 2 digits?
- [ ] Is the day 2 digits?
- [ ] Is there an underscore between the date and title?
- [ ] Are title words separated by hyphens (not spaces)?
- [ ] Is the file in the right folder?
